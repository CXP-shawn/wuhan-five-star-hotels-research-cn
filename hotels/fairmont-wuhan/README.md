## 2. 位置与交通

- **详细地址**:湖北省武汉市江汉区淮海路 249 号
- **所在行政区**:江汉区
- **所在商圈**:武汉中央商务区 CBD、泛海国际
- **地铁交通**:靠近地铁 7 号线王家墩东站 / 取水楼站
- **机场 / 火车站**:距天河国际机场约 25 分钟车程,距汉口站约 5 分钟车程
- **周边地标**:泛海国际 SOHO、江汉路步行街、武汉中央商务区绿地中心

## 3. 核心卖点

- **建筑与设计**:现代新古典奢华设计,含艺术装饰风大堂,2019 年开业为当时江汉 CBD 地标级豪华
- **景观资源**:部分客房享 CBD 天际线景
- **行政酒廊**:Fairmont Gold 行政礼宾层
- **SPA / 泳池 / 健身**:室内泳池、健身房、Spa 中心
- **宴会与会议**:大宴会厅可容纳数百人
- **奖项 / 榜单**:大众点评 5.0 / 携程 5.0(超过 5000 条点评)
