## 项目概述

本仓库系统性收录了武汉市(含江岸、江汉、硚口、武昌、洪山/光谷、汉阳、东西湖、经开沌口、东湖新技术开发区等行政区)所有在营或刚开业的五星级酒店资料,涵盖两类:

1. **中国国家旅游局(现文化和旅游部)官方评定挂牌的五星级酒店**(以湖北省文化和旅游厅 `2024-12-27` 星级旅游饭店名单和武汉市文化和旅游局《星级饭店》名录为准)。
2. **未挂牌但国际品牌自我定位豪华 / 五星 / 超五星的酒店**(万豪 / JW 万豪 / 丽思卡尔顿 / 威斯汀 / 喜来登 / 福朋、希尔顿 / 康莱德、洲际 / 皇冠假日 / voco、凯悦 / 凯悦嘉寓、香格里拉、瑞吉、费尔蒙、万丽、万达瑞华 / 嘉华 / 颐华、东湖宾馆 等)。

> 截至本次截止日期,武汉暂未有公开报道的 **JW 万豪**、**柏悦 / 君悦 / 凯悦嘉寓**、**康莱德**、**华尔道夫**、**洲际 / 瑞吉 / 索菲特 / 文华东方 / 凯宾斯基 / 雅诗阁尊享** 等品牌的独立五星物业;本清单也未纳入 **Novotel / Mercure / Pullman / Dorsett** 等定位四星或非标准五星的酒店。这部分在 `methodology.md` 中有逐条披露。
