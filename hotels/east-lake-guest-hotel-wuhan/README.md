# 武汉东湖宾馆 · East Lake State Guest Hotel

## 2. 位置与交通

- **详细地址**:湖北省武汉市武昌区东湖路 142 号(东湖风景区内)
- **所在行政区**:武昌区
- **所在商圈**:东湖风景区 / 听涛景区
- **地铁交通**:较远,一般建议打车或驱车进入

## 3. 核心卖点

- **建筑与设计**:1953 年建成,是党和国家领导人接待场所。毛泽东 48 次下榻,历史别墅群含梅岭一号;梅岭礼堂可容 1,000 人
- **景观资源**:东湖湖景、园林景
- **行政酒廊**:暂无公开数据
- **SPA / 泳池 / 健身**:泳池、健身、园林
- **宴会与会议**:梅岭礼堂(1000 人以上)、多个中型会议厅
- **奖项 / 榜单**:湖北省文物保护单位(梅岭一号)、中国国宾馆代表之一
