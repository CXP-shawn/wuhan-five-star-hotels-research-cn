# 武汉香格里拉大饭店 · Shangri-La Hotel, Wuhan

> 一句话定位:**武汉最资深的五星国际品牌之一,汉口金融区老牌标杆**

- **所属品牌**:香格里拉 Shangri-La
- **所属集团**:香格里拉集团(Shangri-La Group)
- **星级类型**:国家旅游局官方评定五星(湖北省文旅厅 2024-12-27 名单)
- **开业年份**:1999
- **客房总数**:442
- **客房概要**:442 间客房,含 45 间亲子套房

## 2. 位置与交通

- **详细地址**:湖北省武汉市江岸区建设大道 700 号
- **所在行政区**:江岸区
- **所在商圈**:王家墩 CBD / 新华路金融区
- **地铁交通**:汇聚三条地铁线(3/6/7 号线范湖站等),毗邻万象城
- **机场 / 火车站**:距武汉天河国际机场约 35 公里
- **周边地标**:万象城、武广商圈、建设大道金融街、菱角湖公园

## 3. 核心卖点

- **建筑与设计**:1999 年建成,亚洲经典豪华酒店风格,多次翻修,客房以香格里拉标志性仁爱待客体验为核心
- **景观资源**:部分楼层可俯瞰菱角湖与江城夜景
- **行政酒廊**:香格里拉 Horizon Club 行政酒廊
- **SPA / 泳池 / 健身**:健身中心、室内游泳池
- **宴会与会议**:大宴会厅 1200 平米级,5 米挑高,可分割
- **奖项 / 榜单**:2025 携程口碑榜豪华酒店榜、2025 飞猪年度卓越酒店、2024 美团影响力酒店、2024 高德指南必去榜酒店

### 签名餐厅

- 香宫 Shang Palace — 粤菜及武汉本地名菜(热干面、藕汤、武昌鱼)
- 香苑咖啡厅 Cafe Sha — 自助餐厅,欧陆、亚洲及湖北风味
- 大堂酒廊 The Lobby Lounge

## 6. 其他重要维度

- **近期动态 / 风险点**:建筑年代较早,硬件老化是部分 OTA 差评焦点;基础服务稳定性高

## 7. Pros / Cons / 适合谁

- **Pros**:金融区核心、地铁便捷、粤菜与热干面兼备、老牌忠诚度口碑
- **Cons**:建筑较老、硬件不如新楼盘、停车位在 CBD 相对紧张
- **Fit**:适合:王家墩/武广商务客、Golden Circle 会员。不适合:追求极致新派设计或长江江景的客人
