## 一、概述与分级说明

### 1.1 武汉长江江景背景

武汉地处长江与汉水交汇之处，三镇隔江鼎立，长江穿城而过。汉口沿江大道、武昌临江大道、汉阳滨江大道三条滨江干道汇聚了大量五星级及国际品牌酒店。沿江约25公里岸线覆盖多座历史地标：武汉长江大桥（最知名，连接武昌-汉阳）、鹦鹉洲长江大桥、杨泗港长江大桥、黄鹤楼、晴川阁、龟山电视塔等，夜间长江两岸灯光秀更是全国著名。

### 1.2 江景类型定义

| 级别 | 定义 | 判断标准 |
|------|------|----------|
| **① 一线正江景** | 客房窗户直面长江，无遮挡，可清晰看到江面及对岸 | 官网/品牌页明确标注江景房型；OTA房型描述含"一线江景"；多个用户点评确认 |
| **② 侧江景 / 远眺** | 客房可看到长江，但因角度、距离或楼宇遮挡，视野受限；天气不佳时效果打折 | 官网描述含"江景"但位置距江岸较远；用户反馈"可看到但不清晰"或"天气好才看到" |
| **③ 仅公区江景** | 酒店餐厅、酒吧、大堂、泳池可看江，客房无直接江景 | 房间为城市景观，仅公共区域具备江景描述 |
